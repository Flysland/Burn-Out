# Burn Out – Multiplayer Game (Unreal Engine 5)

## Overview

Burn Out is a **cooperative multiplayer game (up to 4 players)** developed using Unreal Engine 5.
The project focuses on **real-time networking, synchronization, and gameplay systems**.

---

## Objectives

* Develop a real-time multiplayer experience
* Implement a reliable client-server architecture
* Ensure smooth synchronization between players
* Design collaborative gameplay mechanics

---

## Features

* Cooperative gameplay (up to 4 players)
* Real-time client-server communication
* Dynamic task-based gameplay system
* Player interaction and synchronization

---

## Technical Highlights

* Network architecture design (client-server)
* Real-time data synchronization
* Latency management and optimization
* Gameplay system implementation in C++

---

## Technologies

* Unreal Engine 5
* C++
* Networking systems (UE framework)
